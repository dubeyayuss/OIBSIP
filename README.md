# Python & Web Development Projects

This repository contains a collection of hands-on Python and Web Development projects created to strengthen practical programming skills and real-world problem solving. The projects focus on core Python concepts, API integration, and responsive web development.

---

## 📌 Projects Included

### 1️⃣ BMI Calculator (CLI)
- A command-line Python application that calculates Body Mass Index (BMI) based on user input.
- Classifies BMI into standard health categories.
- Demonstrates Python fundamentals, user input handling, and conditional logic.

**Technologies:** Python

---

### 2️⃣ Random Password Generator (CLI)
- Generates secure random passwords based on user-defined length and character preferences.
- Uses Python’s randomization techniques for password creation.
- Focuses on input validation and string manipulation.

**Technologies:** Python

---

### 3️⃣ Weather Application (CLI)
- A command-line application that fetches real-time weather data using a public weather API.
- Displays temperature, humidity, and weather conditions for a given city.
- Implements API integration and JSON data handling.

**Technologies:** Python, REST API, JSON

---

## 🚀 How to Run Python Projects
1. Clone this repository  
2. Navigate to the required project folder  
3. Run the Python file using: python filename.py
